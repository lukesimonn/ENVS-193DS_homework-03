# ENVS-193DS_homework-03

## General Information

This repo contains all of my work to complete homework 3 for my environmental statistics class at UCSB for the Spring 2026 quarter.

## Data and file Information

The code folder contains all code (in the respective qmd file) needed for this assignment. It also contains the rendered pdf file. The images sub-folder contains all needed images to render the code file.

The data folder includes two required csv files for the code. The first of which is a hypothetical Temperature and Kelp data frame used in the first problem of the homework. The second, is a data frame created be me containing personal observations used in problem 2 and referenced in problem 3.
